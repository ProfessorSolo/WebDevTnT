# **The point from wence transformation occureth**

Transforms don’t just _happen_ — they pivot from a specific point.

Move that pivot, and the entire motion changes character.

---

<p class="ps-caption">Pivot at the top → classic dropdown hinge.</p>

```css
.monkey {
  transform-origin: top center;
  transform: rotateX(45deg);
}
```

## Keyword Origins

You can also use keywords:

```css
transform-origin: top left;
transform-origin: bottom center;
transform-origin: right;
```
