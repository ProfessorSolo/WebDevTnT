# **Glide, Bounce, Mirror, Escape**

Here’s where transforms and transitions finally shake hands.

Transforms give us **poses**.  
Transitions give us **flow**.

Each recipe below can be part of the show.

---

<div class="ps-recipes">

  <section class="ps-recipe">
    <h3 class="ps-recipe-title">1) Glide</h3>
    <p class="ps-note">A soft entry from the side. Great for cards, modals, and reveals.</p>
    <div class="ps-stage ps-glide">
      <img class="ps-kitty" src="/assets/lessons/m5-space-kitty-in-space-256w.webp" alt="Space kitty gliding in smoothly." />
    </div>

```css
.card {
  opacity: 0;
  transform: translateX(-40%) scale(0.95);
  transition: transform 650ms ease-out, opacity 650ms ease-out;
}
.card:hover {
  opacity: 1;
  transform: translateX(0) scale(1);
}
```

  </section>

  <section class="ps-recipe">
    <h3 class="ps-recipe-title">2) Bounce</h3>
    <p class="ps-note">A playful hover hop. Perfect for buttons and icons.</p>
    <div class="ps-stage ps-bounce">
      <img class="ps-kitty" src="/assets/lessons/m5-space-kitty-in-space-256w.webp" alt="Space kitty bouncing upward on hover." />
    </div>

```css
.icon {
  transition: transform 520ms cubic-bezier(0.2, 0.9, 0.2, 1.4);
}
.icon:hover {
  transform: translateY(-26px) scale(1.07);
}
```

  </section>

  <section class="ps-recipe">
    <h3 class="ps-recipe-title">3) Mirror</h3>
    <p class="ps-note">A clean Y-axis flip. Use for toggles, switches, and card backs.</p>
    <div class="ps-stage ps-mirror">
      <img class="ps-kitty" src="/assets/lessons/m5-space-kitty-in-space-256w.webp" alt="Space kitty flipping like a mirrored card." />
    </div>

```css
.tile {
  transition: transform 700ms ease-in-out;
  backface-visibility: hidden;
}
.tile:hover {
  transform: rotateY(180deg);
}
```

  </section>

  <section class="ps-recipe">
    <h3 class="ps-recipe-title">4) Escape</h3>
    <p class="ps-note">Controlled chaos. A fast launch out of frame.</p>
    <div class="ps-stage ps-escape">
      <img class="ps-kitty" src="/assets/lessons/m5-space-kitty-in-space-256w.webp" alt="Space kitty escaping diagonally with rotation and scale." />
    </div>

```css
.thing {
  transition: transform 800ms ease-in, opacity 800ms ease-in;
}
.thing:hover {
  transform: translate(70%, -45%) rotate(14deg) scale(1.18);
  opacity: 1;
}
```

  </section>

</div>
