## Forms - Introduction

![alt_text](/assets/lessons/m2-forms-intro-card.png 'image_tooltip')

Forms allow users to input and submit data for collection and processing.

We use forms for:

- Collecting information from users
- Logging in to a site
- Performing calculations
- Standardizing contact requests
- Managing website content
- Purchasing items online
- Search
