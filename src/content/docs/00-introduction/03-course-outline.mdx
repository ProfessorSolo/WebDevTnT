## **Web Development Tools and Techniques**

### **Course Description**

​Students will develop an understanding of web development processes as part of a team of developers and project stakeholders. Modern HTML and CSS technologies that allow for the creation of advanced responsive layouts and animated user interfaces will be explored. Students will discuss and practise sharing and version-controlling code with a team of developers, automating web development tasks using task runners and improving CSS development using the latest CSS pre-processors.​

### **Course Learning Outcomes/Competencies**

Upon successful completion of this course, the student will be able to:

- Create advanced responsive layouts using modern CSS.
- Create animated user interface elements.
- Write code for delivering optimized images to phones, tablets and desktop devices.
- Write CSS using a CSS pre-processor.
- Perform version control and share code with other developers.
- Automate web development tasks using task-runner tools.
- Develop a web site as part of a team of developers .

### **Learning Resources**

- Windows or Mac Laptop
- Modern Internet Browser - Lectures will be conducted in Chrome
- Visual Studio Code
- Access to AI system (ChatGPT, Claude, Gemini, or similar) recommended, but not required
- Personal GitHub Account (free)

### **Course Goals**

Upon successful completion of this course students will be able and develop modern, responsive, and visually engaging web applications using advanced CSS techniques. Students will be introduced to utility-first post-processing and CSS pre-processors as well as task automation via build tools. They will gain practical experience in version control for collaborative workflows which is essential for managing team-based projects. Students will refine their ability to create optimized images, animated interfaces, and maintainable codebases as well as building accessible HTML5 forms to ensure inclusive user experiences. This course further emphasizes performance strategies for delivering content across devices, as well as teamwork and communication through group presentations and discussions, preparing students to contribute effectively in professional web development environments.
