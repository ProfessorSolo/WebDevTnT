# Meet the Variables

Before you can cast spells, you need ingredients.  
Variable ingredients start with `--`, live anywhere, and cascade like any other CSS property.

---

## ğŸ”® Declaring a Custom Property

```css
:root {
  --brand-color: #ff6ec7;
}
```

A variable is just a named value stored on an element.  
Putting them in `:root` makes them global.
