# Scope the Magic

Custom properties inherit.  
This makes them perfect for **component-level overrides**, theme sections, UI modes, and nested spellcasting.

```css
}

  --font-heading: "Permanent Marker";
  --accent: #00b3ff;

```

Every element inside `.alt-theme` will inherit the new values.
