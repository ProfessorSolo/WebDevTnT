## The Syntax

```css

```

That means:

- Make **4** columns
- Each column is **1fr**

Same result. Less noise.

## The Takeaway

Use `repeat()` when:

- Your grid has repeated structure
- You want to scale column/row counts quickly
- You want templates that stay readable under pressure
