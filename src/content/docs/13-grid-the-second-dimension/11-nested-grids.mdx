## The Pattern

Outer grid defines the big structure.

Inner grid defines the structure *inside a region*.

```css
.page{
  display: grid;
  grid-template-columns: 14rem 1fr;
  grid-template-areas:
    "header  header"
    "sidebar content";
}

.content{
  grid-area: content;
  display: grid;              /* nested grid */
  grid-template-columns: repeat(3, 1fr);
  gap: 1rem;
}
```

No magic.

Just composition.
