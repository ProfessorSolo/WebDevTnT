## How Grid Decides What Goes Where

When you don‚Äôt explicitly place items, Grid makes choices.

Those choices are governed by **auto-placement** ‚Äî specifically `grid-auto-flow`.

This is the rulebook Grid uses to fill the grid.

---

## The Default: Row Flow

```css
grid-auto-flow: row;
```

This is the default behavior:

- Fill rows left to right
- Then create new rows as needed

It feels familiar ‚Äî but it‚Äôs still Grid, not Flexbox.

---

## Column Flow Changes Everything

```css
grid-auto-flow: column;
```

Now Grid:

- Fills top to bottom
- Then creates new columns

Same items.  
Same grid.  
Very different outcome.

---

## Dense Packing (Use Carefully)

```css
grid-auto-flow: row dense;
```

`dense` tells Grid to backfill gaps when possible.

This can:

- Reduce empty space
- Reorder items visually

Powerful ‚Äî but potentially confusing if overused.
