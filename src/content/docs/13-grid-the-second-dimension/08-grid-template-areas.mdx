## Blueprints, Not Coordinates

Line numbers are precise.

---

## Define Areas on the Container

```css
.layout {
  display: grid;
  grid-template-columns: 14rem 1fr;
  grid-template-rows: auto 1fr auto;

  grid-template-areas:
    'header  header'
    'sidebar content'
    'footer  footer';
}
```

The strings describe the grid.

---

## Assign Items to Areas

```css
header {
  grid-area: header;
}
aside {
  grid-area: sidebar;
}
main {
  grid-area: content;
}
footer {
  grid-area: footer;
}
```

Now your HTML can stay semantic, and the grid does the layout work.

---
