# **The Key to Animation**

Alright ‚Äî time to move beyond A ‚Üí B motion.

With **@keyframes**, we design **the entire journey**, not just the endpoints.

Transitions react.  
**Animations perform.**

---

## üé¨ Declaring a Keyframe

A keyframe block defines _how_ something changes over time.

```css
@keyframes float {
  0% {
    transform: translateY(0);
  }
  50% {
    transform: translateY(-12px);
  }
  100% {
    transform: translateY(0);
  }
}
```

This is a full motion arc ‚Äî up, then down, smooth loop.

---

## üéõ Attaching an Animation

```css
.orb {
  animation: float 2s ease-in-out infinite;
}
```

Order matters:

`animation: name duration timing-function iteration-count;`

That‚Äôs the core anatomy.

---

## üß© Using Keywords Instead of Percentages

You can also use:

```css
from {
  opacity: 0;
}
to {
  opacity: 1;
}
```

Equivalent to `0%` ‚Üí `100%`.  
Perfect for simple fade or slide sequences.

---

## üéö Multi-Step Sequences

Keyframes can have _as many steps as you want_:

```css
@keyframes spinPop {
  0% {
    transform: scale(0.8) rotate(0deg);
  }
  30% {
    transform: scale(1) rotate(180deg);
  }
  70% {
    transform: scale(1.15) rotate(270deg);
  }
  100% {
    transform: scale(1) rotate(360deg);
  }
}
```

This is how we shape character and personality ‚Äî dynamic, playful, dramatic.
