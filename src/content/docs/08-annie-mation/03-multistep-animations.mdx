# **Step 2, 3, Step, 2, 3, Dip**

Now we level up - again.

We've seen simple 0% ‚Üí 100% transitions.

But the real magic happens when we break the journey into **multiple beats**, each with its own personality.

Multi‚Äëstep animations let us:

- Stage reveals
- Build anticipation
- Add overshoots and rebounds
- Create ‚Äústories‚Äù inside motion
- Sync transforms, color shifts, opacity, and more

This is where AnnieMation takes the dancefloor.

## üé≠ Three‚ÄëBeat Example

```css
@keyframes revealBoom {
  0% {
    opacity: 0;
    transform: scale(0.8) rotate(-8deg);
  }
  40% {
    opacity: 1;
    transform: scale(1.05) rotate(3deg);
  }
  100% {
    opacity: 1;
    transform: scale(1) rotate(0deg);
  }
}
```

Small ‚Üí big ‚Üí settle.  
Tension ‚Üí anticipation ‚Üí release.

## üéö Complex Motion Arc

```css
@keyframes orbit {
  0% {
    transform: rotate(0deg) translateX(0);
  }
  25% {
    transform: rotate(90deg) translateX(20px);
  }
  50% {
    transform: rotate(180deg) translateX(0);
  }
  75% {
    transform: rotate(270deg) translateX(-20px);
  }
  100% {
    transform: rotate(360deg) translateX(0);
  }
}
```

This establishes rhythm ‚Äî a repeatable _pattern of motion_, not just a spin.

---

## üß© Coordinate Multiple Properties

Opacity, transforms, shadows, colors ‚Äî all can live inside the same arc.

```css
@keyframes glowLift {
  0% {
    opacity: 0;
    transform: translateY(12px);
    box-shadow: 0 0 0 rgba(0, 0, 0, 0);
  }
  50% {
    opacity: 1;
    transform: translateY(-4px);
    box-shadow: 0 4px 14px rgba(0, 0, 0, 0.25);
  }
  100% {
    opacity: 1;
    transform: translateY(0);
    box-shadow: 0 2px 8px rgba(0, 0, 0, 0.15);
  }
}
```

That‚Äôs animation as **design language**.
